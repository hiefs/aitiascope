The plan for documentation

Discussions

As a public forum board, Aitiascope uses discussions as the public open forum for community members. You can join with a personal github account.

Wiki

Github also has the availability for a wiki to be kept. Though the wiki currently is empty, a system will be put in place to add, update, an upkeep the information there.
